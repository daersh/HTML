src 속성에 경로를 설정하여 이미지 파일을 불러온다.

자주 사용하는 속성

alt 속성

화면 낭독기는 이미지를 읽을 수 없기 때문에 대체되는 텍스틀를 대신 읽어준다.
혹은 사진의 경로가 잘못되거나 이미지를 제대로 표현할 수 없는 경우 대체 텍스트의 용도로도 사용 가능하다.

width, height

사진의 높이와 너비를 지정할 수 있다.
고정된 단위의 크기로도 지정 가능하며
가변 크기 단위의 크기로도 지정 가능하다.

고정 크기 단위: 화면 사이즈가 달라져도 크기가 변하지 않는다.

가변 크기 단위: 화면 사이즈에 따라 변경된다.

이미지 구역을 2개로 나누어 각각의 링크 설정
